# Network Monitoring Dashboard

## Project Overview
A web-based network monitoring dashboard built with Python and Flask. 
It monitors network devices in real-time using Ping and displays their status in a clean web interface.

## Features
-  Real-time device monitoring via Ping
-  Auto-refresh every 30 seconds
-  Add new devices to monitor
-  Delete devices from the list
-  Export monitoring report as JSON
-  Clean web interface built with Flask

## Technologies Used
- Python 3.13
- Flask 3.1
- ping3
- JSON
- HTML/CSS

## How to Run
1. Install requirements:
pip install flask ping3

2. Run the script:
python network_monitor.py

3. Open browser:
http://localhost:5000
